# Установка Node.js для запуска проекта

Сообщение `command not found: npm` означает, что **Node.js не установлен** на вашем Mac.

## Способ 1: Установщик с сайта (рекомендуется)

1. Откройте в браузере: **https://nodejs.org**
2. Скачайте версию **LTS** (зелёная кнопка).
3. Запустите скачанный `.pkg` и пройдите установку.
4. **Закройте и снова откройте терминал** (или Cursor).
5. В папке проекта выполните:

```bash
cd /Users/vladimir/Desktop/weding
npm install
npm run dev
```

6. Откройте в браузере: **http://localhost:3000**

---

## Способ 2: Через Homebrew (если уже установлен)

Если у вас установлен Homebrew:

```bash
brew install node
```

После установки — снова `npm install` и `npm run dev` в папке проекта.

---

## Проверка установки

После установки Node.js выполните в терминале:

```bash
node -v
npm -v
```

Должны появиться версии (например `v20.x.x` и `10.x.x`).
