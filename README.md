# Fit Track — Тренировки 90 дней

Простое веб-приложение для фитнеса. Открой ссылку на телефоне — работает как сайт, прогресс сохраняется в браузере.

## Стек

- **Next.js 14** (App Router)
- **React 18**, **TypeScript**
- **Tailwind CSS** — тёмная тема, простой интерфейс
- **Lucide React** — иконки
- **date-fns** — даты
- **LocalStorage** — прогресс не пропадает

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Логика тренировок (90 дней)

- **Дни:** Пн, Ср, Пт.
- **Исходные максимумы:** подтягивания 7, приседания 40, пресс 30, отжимания 30.
- **Прогрессия:** +2–5% к объёму в неделю (прогрессивная перегрузка).
- **Чередование:** сила → выносливость → взрывная.
- **Проверка максимума:** раз в 2 недели (каждые 14-й тренировочный день).

## Функционал

1. **Dashboard** — сегодняшняя тренировка, уровень/XP, кольца активности, статистика.
2. **Лог** — ввод повторений по подходам, сохранение тренировки.
3. **Достижения** — бейджи (первая сотня приседаний, король турника, 30 дней без пропусков и др.).
4. **История** — календарь с отметками выполненных тренировок.
5. **Геймификация** — уровни и XP за каждую тренировку и достижения.

## Дизайн

- Тёмная тема, минимум декора.
- Адаптивно под телефон: крупные кнопки, читаемый текст.
- Открой ссылку на Vercel с телефона — сохрани в закладки или на главный экран.

Данные в **LocalStorage** — прогресс в твоём браузере.

## Деплой на Vercel

Бесплатный хостинг для Next.js. Репозиторий уже на GitHub — остаётся подключить его к Vercel.

### Шаг 1: Войти в Vercel

1. Зайди на **https://vercel.com**
2. Нажми **Sign Up** и войди через **GitHub** (так Vercel получит доступ к репозиториям).

### Шаг 2: Импорт проекта

1. На главной Vercel нажми **Add New…** → **Project**
2. В списке репозиториев найди **NG-Vladimir/sport** и нажми **Import**
3. Оставь настройки по умолчанию:
   - **Framework Preset:** Next.js (определится сам)
   - **Root Directory:** `./`
   - **Build Command:** `npm run build` (или пусто — по умолчанию)
   - **Output Directory:** не трогай (для Next.js не нужен)
4. Нажми **Deploy**

### Шаг 3: Готово

Через 1–2 минуты сборка закончится. Vercel выдаст ссылку вида:

**https://sport-xxx.vercel.app**

Каждый пуш в ветку `main` на GitHub будет автоматически собираться и деплоиться.
